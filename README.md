# Project Name

E-Commerce Web Application

## Overview

This project is a modern web application built using the latest frontend technologies. It provides a fast, responsive, and user-friendly interface for delivering a seamless user experience.

## Features

* Fast and optimized performance using Vite
* Responsive UI design
* Reusable and scalable components
* Clean and maintainable code structure
* Modern UI with Tailwind CSS and shadcn components

## Tech Stack

* **Frontend:** React, TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **UI Components:** shadcn-ui

## Project Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone -b mohit_dev https://github.com/mohitjoshi29/webiators_ecommerce_web.git

# Navigate to project folder
cd webiators_ecommerce_web

# Install dependencies
npm install

# Start development server
npm run dev
```

## Build for Production

```bash
npm run build
```
